MundoPet

Este projeto foi desenvolvido como parte do currículo do curso técnico em Desenvolvimento de Sistemas, com o propósito de aplicar conceitos fundamentais de Web Design e Front-end.

O foco principal foi o exercício da estrutura semântica, estilização moderna e, acima de tudo, a colaboração e fluxo de trabalho em equipe utilizando Git.


🎯 Objetivos do Projeto
Consolidação de Conhecimento: Praticar a escrita de código limpo em HTML5 e CSS3.

Responsividade: Criar uma interface que se adapte a diferentes tamanhos de tela (Mobile First).

Trabalho em Equipe: Utilizar metodologias para divisão de tarefas e integração de código via GitHub.

UI/UX: Aplicar princípios básicos de usabilidade e hierarquia visual.


🛠️ Tecnologias Utilizadas
HTML5: Estruturação semântica do conteúdo.

CSS3: Estilização, Layouts (Flexbox/Grid) e animações.

Git & GitHub: Controle de versão e colaboração.
